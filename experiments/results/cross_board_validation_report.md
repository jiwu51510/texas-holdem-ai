# 跨公共牌双维度胜率-策略验证实验报告

## 实验目的

验证：**固定一个手牌，在不同公共牌情况下：**
当以下两个条件同时满足时，策略是否相同？
1. 手牌vs对手范围的胜率相近（差异<5%）
2. 自己范围vs对手范围的胜率相近（差异<5%）

## 实验方法

1. 固定一个手牌（如AhKh）
2. 生成多个不同的公共牌场景
3. 对每个场景：
   - 从对手范围中去除与我手牌冲突的组合
   - 计算手牌vs对手范围的胜率
   - 计算范围vs范围的胜率
   - 使用CFR求解最优策略
4. 找出双维度胜率都相近的不同公共牌场景
5. 比较这些场景下的策略是否相同
